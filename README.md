# AI Harness Engineer Roadmap

A lightweight career-planning web app for scoring the core skill areas behind AI harness engineering and generating a simple 30-day plan.

## What works
- Rate yourself across five skill areas
- Generate a readiness score
- Produce a practical 30-day learning roadmap based on weak spots

## Run locally
```bash
python3 -m http.server 8000
# then open http://localhost:8000/index.html
```

## Meaningful user path
1. Score your current skills
2. Generate the roadmap
3. Use the four-week plan as your first curriculum

## Known gaps
- No account system or saved plans yet
- Skill framework is opinionated and lightweight
- No direct job-board or curriculum integrations yet
